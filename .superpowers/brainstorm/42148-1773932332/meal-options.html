Napi étkezések — melyik opciókat mutassuk?

A varázslóban jelenleg 2/3/4/5 étkezés közül lehet választani. Az IF (időszakos böjt) opciókat hol és hogyan mutassuk?

A

Beépítve a listába — 5+2 opció

A varázslóban mind az 5 opció (+ IF 16:8 + IF 18:6) egy helyen, sorban:

🍽️ 2 étkezés – Reggeli + Vacsora
🍽️🍽️ 3 étkezés – Reggeli + Ebéd + Vacsora (alapértelmezett)
🍽️🍽️🍽️ 4 étkezés + Tízórai
🌟 5 étkezés + Tízórai + Uzsonna
⏱️ IF 16:8 – 8 órás evési ablak (pl. 12:00–20:00)
🌙 IF 18:6 – 6 órás evési ablak (pl. 13:00–19:00)

Előny

  • Minden egy helyen
  • Egyszerű

Hátrány

  • Hosszabb lista
  • IF keveredik a "normál" étkezésekkel
B

Szekció bontás — Normál / Időszakos böjt

A listát két részre osztjuk egy szeparátorral:

── Napi étkezések ──
🍽️ 2 étkezés / 3 étkezés / 4 étkezés / 5 étkezés

── Időszakos böjt ──
⏱️ IF 16:8 – Napi 2 étkezés, 8 órás ablakban
🌙 IF 18:6 – Napi 1 étkezés, 6 órás ablakban

Előny

  • Vizuálisan tiszta
  • IF opciók külön, magyarázattal

Hátrány

  • Kicsit több hely
C

Toggle gomb — "Időszakos böjt" kapcsoló

A normál 2/3/4/5 opció mindig látható. Egy kis "Időszakos böjtöt szeretnék" toggle gomb elrejti a 2-5-öt és mutatja az IF opciókat.

📌 Alapértelmezés: normál étkezések
📌 Toggle → IF 16:8 és IF 18:6 jelenik meg

Előny

  • Nem tolja meg a UI-t
  • Egyértelmű szándék-váltás

Hátrány

  • IF egy extra kattintást igényel

Kattints a preferált opcióra, vagy írd meg a terminálban.
